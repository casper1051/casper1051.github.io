What It Does

ArityOS is a customized version of Shimboot, but has wider support for some boards like dedede, bork, and octopus. Promarily designed for the R853- series(as well as the Borkbook), ArityOS can game Minecraft on 60+FPS, and shaders on ~15 FPS.

How It Works

The tool uses a chainloading system to boot different partitions. There are in total ~10 partitions, which makes the part layout: Stateful, Kernel, Fallback Bootloader, Recovery, User space(expand as needed), ChromeOS booter, ChromeOS patched, Primary Bootloader, USB Bootloader, Board switcher, ChromeOS Recovery in case something broke, State Testing revert scripts, Troll part to say "ChromeOS is missing or damaged". Everything that changes per use case is in the kernel partition, which can be switched out by booting the board switching partition.

Tech Stack

- Language: Bash, Shell
- Libraries: frecon-lite
- Tested on: Dedede, Octopus, Nissa, Bork*

*Bork is a customized board.
