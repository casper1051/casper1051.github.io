My Projects

ArityOS

What it does: Allows for more flexible modification of chromebooks, as well as including a boot-from-usb package.
How it works: Uses a modified ChromeOS Developer Shim, as well as calling exec init from the first loader.

Liberia

What it does: Provides a clean front-end library checkout menu for some OPALS pages.
How it works: Electron-based application using puppeteer to avoid API call intercepts on the back-end, a modded distro of ArityOS for the bootloader, paired with Openbox and autostart.

NoGuardian

What it does: Provides a way to circumvent initial ChromeOS sync downloads, including extensions.
How it works: Fills up the internal eMMC in the Chromebook after a powerwash(to delete existing data), interrupting the sync cycle.

0gUI

What it does: ok this one was just me raging at KIPR's AP mode not detecting in macOS
How it works: sudo rm /usr/local/bin/botui

InstaBork

What it does: Instantly bricks ChromeOS
How it works: Creates a while true reload loop, instantly freezing the computer until it is force shut down.

BoardSwapper

What it does: Allows some RMA-based ChromeOS utilities to be used on other boards
How it works: In most cases, only the kernel (on part 2) is board-specific. If we switch out the kernel with a new one, that board will see the shim as valid. This means a dedede shim with an octopus kernel will boot.
