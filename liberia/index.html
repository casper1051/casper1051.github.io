Liberia

What It Does

Liberia is an Electron-based application paried with a modified operating system image that serves as a self check-out system for libraries using an OPALS based system.

How It Works

The tool uses puppeteer to communicate with a headless browser running OPALS pages. Liberia must be pre-programmed to operate with specific URL's and authentication systems. Everything that changes per use case is in the scripts.js file, which controls all behavior.

Tech Stack

- Language: HTML, CSS, JS, NodeJS, Bash
- Libraries: npm, nodejs, puppeteer, electron
- Tested on: Debian Linux, macOS
