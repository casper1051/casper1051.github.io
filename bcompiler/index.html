BCompiler

What It Does

BCompiler is a NodeJS based API endpoint for uploading promarily JavaScript-based programs to the KIPR Wombat.

How It Works

While not a conventional 'compiler', the tool makes use of fetch() requests to be able to receive code from other devices connected to the Wombat. A program saving option can be implemented, and so can support for other languages. It uses HdiUi:Interface, so any language capable of creating subprocesses should be compatible.

Tech Stack

- Language: NodeJS, Bash
- Libraries: Express, HdiUi:Interface
- Tested on: Hosted on Debian Bullseye (ie WombatOS), but should be compliant with any modern OS.
